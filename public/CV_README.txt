README: CV File Setup
======================

Please add your CV file to this directory with the name: cv.pdf

The portfolio website expects a file named "cv.pdf" in the /public/ folder.

Steps:
1. Export your CV as a PDF file
2. Rename it to: cv.pdf
3. Place it in this /public/ directory
4. Delete this README file

The CV download buttons are located in:
- Hero section (top CTA)
- Contact section (bottom CTA)
- Footer (center link)

All buttons will link to: /cv.pdf
