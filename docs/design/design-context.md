# Design Context Specification

Reference style inspired by https://www.pszostak.pl

## Design Philosophy

Clarity over decoration. Content over interface.

The UI should remain visually quiet so that portfolio projects are the
primary focus.

## Style Keywords

minimal clean editorial grid-driven typography-first high whitespace
professional

## Layout System

Use a 12-column responsive grid.

Desktop: max-width: 1200--1400px columns: 12 gutter: 24--32px

Tablet: columns: 8

Mobile: columns: 4

## Spacing System

Spacing scale: 4 8 12 16 24 32 48 64 96 128

Use generous vertical spacing between sections.

## Typography

Use modern neo‑grotesque sans-serif fonts.

Examples: Inter Helvetica Neue Haas Grotesk Söhne Switzer

Hierarchy example: H1 48--64 H2 32--40 H3 24--28 Body 16--18 Small
12--14

Line height: 1.4--1.6
