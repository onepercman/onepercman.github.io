# Component Rules

## Navigation

Minimal top navigation. No heavy backgrounds.

Items: Work About Contact

## Project Card

Structure: Image Title Short description Meta info (year or role)

Hover: Subtle scale or opacity change.

## Buttons

Style: Simple outline or text button.

Padding: 12px 20px

## Section Layout

Each section should have generous vertical spacing.

Typical padding: 80px--120px vertical.
